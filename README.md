# 🕹️ Basit 2D Platform Oyunu

Bu proje, HTML5 **Canvas** ve **JavaScript** kullanılarak geliştirilmekte olan temel bir 2D platform oyunudur. JavaScript öğrenme sürecimin bir parçası olarak geliştirilmiştir.

## 🚀 Özellikler
* **Karakter Kontrolü:** Sağ/Sol hareket ve Zıplama (Geliştirilme aşamasında).
* **Platformlar:** Üzerinde durulabilir zeminler.
* **Basit Fizik:** Yer çekimi ve çarpışma algılama mantığı.

## 🛠️ Kullanılan Teknolojiler
* **HTML5:** Oyunun iskeleti ve `<canvas>` alanı için.
* **CSS3:** Oyun alanının görsel düzeni için.
* **JavaScript:** Oyun mantığı, karakter hareketleri ve fizik kuralları için.
